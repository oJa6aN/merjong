# Merjong

This project is currently under development.

## Usage

```html
<div class="merjong">
    55"0m-5'5''55s-55'0''5p-X5"Xz
</div>
```

## About

Merjong is a JavaScript-based image generation tool that uses MPSZ algebraic notation to create images of mahjong tiles.
